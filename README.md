# dani-settings

Personal VS Code configuration and extension pack.

## Features

- **Extension Pack**: Includes a curated set of extensions for development (Vim, Gruvbox, Python, etc.).
- **Recommended Settings**: Command to apply a balanced set of editor configurations.
- **Recommended Keybindings**: Command to merge specialized keybindings for a streamlined workflow.

## Commands

This extension contributes the following commands:

* `Dani: Apply settings`: Applies the recommended global settings.
* `Dani: Apply keybindings`: Appends recommended keybindings to your user profile.

## Requirements

The extension depends on several others (like `vscodevim.vim` and `jdinhlife.gruvbox`) which will be installed automatically as part of the extension pack.
